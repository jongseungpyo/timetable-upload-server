학생을 위한 맞춤 시간표

타임빌더는 학생들이 자신에게 맞는 학원 수업을 쉽게 찾고 관리할 수 있도록 도와주는 플랫폼입니다. 학원과 강사님들의 참여로 더욱 풍부한 시간표 정보를 제공합니다.

맞춤 검색

학교, 학년, 과목별로 나에게 맞는 수업을 쉽게 찾아보세요.

시간표 관리

나만의 시간표를 만들고 충돌 없이 수업을 배치하세요.

실시간 업데이트

새로운 수업과 변경사항이 실시간으로 반영됩니다.

학원 운영자이신가요?

타임빌더에 학원의 시간표 정보를 등록하고 더 많은 학생들과 만나보세요

시간표 제출

학원/강사님의 강의 시간표를 제출해주세요. 검토 후 타임빌더 앱에 반영됩니다.

제출 요구사항

작성 가이드 다운로드 엑셀 양식 다운로드
  • 엑셀 양식 사용: 작성 가이드를 먼저 확인하고, 엑셀 양식을 다운로드하여 작성 후 CSV로 저장하여 제출하세요
  • CSV 파일 형식으로 제출 (Excel에서 "다른 이름으로 저장" → CSV 선택)
  • 강사명, 과목, 대상 학교, 주제, 출강 학원 등 필수 정보 포함
  • 시간표는 일주일 단위로 작성 (일요일~토요일)
  • 업로드 후 웹 테이블에서 데이터 확인/수정 가능

시간표 제출 방법

학원 정보 (자동 입력)

학원명: 로그인 후 자동 입력
담당자: 로그인 후 자동 입력
이메일: 로그인 후 자동 입력

학원 홈페이지, 네이버 블로그, 또는 학원 정보를 확인할 수 있는 공식 링크를 입력하세요.

CSV 파일을 선택하세요
업로드하면 이 영역이 편집 가능한 테이블로 변환됩니다

각 행은 하나의 강의를 의미합니다. 같은 강의의 여러 시간대는 별도 행으로 입력하세요.
